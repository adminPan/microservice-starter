# 拍拍贷微服务starter

### starter简介

    拍拍贷微服务starter是基础框架部研发的一款基于spring-boot的starter。
    开发人员只需在工程中引入microservice-starter的jar包，编写一个main方法，即可启动应用。

### starter包含：

1. spring-boot基础包的引用

2. 基础框架团队开发的二方包

3. 标准的三方包

4. 基础框架团队开发的UTIL工具类

### 注意项：

##### 应用中的必须包含内容：

1. spring boot环境配置中，且其中必须包含com.along101.appId和spring.application.name两个配置项。
